Parasite

Create and Download a Dataset
Parasite is a perpetual git & github scrapper that maintains a datastore of downloaded contents and continuously updates the items with new information as downloaded from the upstream sources. This repository consists of the updater application itself and a Rust library that can be used to read the datastore.

Djanco

Engine for Running Queries
A mostly-in-memory database with a query Rust interface. Handles user-written queries that are executed by the Djanco in-memory database. For each query, Djanco determines the required portion of the datastore, it loads it and executes the query. Queries run with all project metadata in-memory while the content of the source code files remains on disk.
